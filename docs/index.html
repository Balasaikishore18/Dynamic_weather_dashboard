📌 Overview

The Dynamic Weather Dashboard is a web application built using JavaScript, OpenWeatherMap API, and Geolocation API. It provides real-time current weather and a 5-day forecast for any city worldwide or based on the user’s current location.

⚙️ Installation

  1. Clone this repository: 
    git clone https://github.com/your-username/weather-dashboard.git
  2. Navigate into the project folder: 
    cd weather-dashboard
  3. Open index.html in your browser, or use a local server (recommended).
  4. Replace the API key placeholder in script.js: 
    const API_KEY = 'YOUR_OPENWEATHERMAP_API_KEY';

🎮 Usage

  • Enter a city name and click Search.
  • Or click Current Location to fetch weather using geolocation.
  • Dashboard displays:
    • 🌡️ Current temperature
    • 🌫️ Feels like temperature
    • 💧 Humidity
    • 🌬️ Wind speed
    • 📉 Min / Max temperatures
    • 📊 5-day forecast
